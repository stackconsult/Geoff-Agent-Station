# Welcome to Tolaria

This is your personal knowledge management system.

## Features

- **Note Editor** - Write and edit your thoughts
- **Search** - Find notes instantly
- **Auto-Git** - Automatic version control
- **File Watcher** - Detect external changes

## Getting Started

1. Select your Obsidian vault
2. Browse notes in the sidebar
3. Click a note to edit
4. Your changes auto-save
